# Master Collection Companion

Chrome extension companion for Master Collection browser-side Webflow Designer utilities.

## Documents

- `docs/INSTALL.md` - user/tester installation instructions for the unpacked extension.
- `docs/CHROME-WEB-STORE-PUBLISHING.md` - checklist for turning this into an official Chrome Web Store extension.

## Current Extension

- Name: Master Collection Companion
- Manifest: V3
- Scope: Webflow Designer URLs only
- Main features: Paste Guard and interaction cleanup tooling
